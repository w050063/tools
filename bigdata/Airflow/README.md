# 概述
Airflow是一个以编程方式创作，安排和监控工作流程的平台。

使用气流将工作流作为任务的有向非循环图（DAG）。气流调度程序在遵循指定的依赖关系的同时在一组工作程序上执行您的任务。丰富的命令行实用程序可以轻松地在DAG上执行复杂的手术。丰富的用户界面使您可以轻松地可视化生产中运行的管道，监控进度以及在需要时解决问题。

当工作流被定义为代码时，它们变得更易于维护，可版本化，可测试和协作。
