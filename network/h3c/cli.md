# 常用命令列表
``` text
<H3C>                         # 默认进入用户视图
sysname device_names          # 为设备命名为device_names
display current-configuration # 显示当前配置
system-view                   # 进入系统视图模式(配置视图)
undo shutdown                 # 启用以太网端口
shutdown                      # 关闭以太网端口
quit                          # 退出当前视图模式
display ip routing-table      # 显示当前路由表
reset saved-configuration     # 恢复出厂配置，并重启
reboot                        # 重启
save                          # 保存当前配置
```
