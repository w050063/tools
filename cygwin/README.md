## Cynwin部署安装
- 安装源：http://www.cygwin.cn or http://mirrors.aliyun.com/cygwin/
- 安装包选择
  - gcc
  - gcc-c++
  - autoconf
  - automake
  - git
  - curl
  - wget
- 设置系统变量
- 安装[apt-cyg](https://github.com/transcode-open/apt-cyg)
``` bash
wget https://raw.githubusercontent.com/transcode-open/apt-cyg/master/apt-cyg
install apt-cyg /bin/
```
