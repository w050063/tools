#### # 压力测试工具

> yum install siege

#### # 例子

    siege http://118.31.78.77:4003/getIp -c 100 -r 10

    siege  http://www.joedog.org/ POST haha=papa&dada=mama

#### # 参数

    -c 并发数
    -r 循环此处
    -f 使用文件
    -A "string", --agent="string"
    -H "header: value", --header="Header: value"
    -m "string", --mark="string"
