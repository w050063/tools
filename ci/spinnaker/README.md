# 概述

企业级持续交付，快速、安全可重复的部署。

- 官网地址：https://www.spinnaker.io/
- 概念：https://www.spinnaker.io/concepts/

Spinnaker是一个开源的多云持续交付平台，可帮助您以高速度和自信度发布软件更改。

Spinnaker提供两个核心功能：
- 应用管理
- 应用部署

## 应用管理 Application management
- Application
- Cluster
- Server Group
- Load Balancer
- Firewall

## 应用部署 Application deployment
- Pipeline
- Stage
- Deployment strategies
